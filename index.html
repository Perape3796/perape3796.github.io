# 📊 TelAnalysis - Turn telegram chats into visual reports
Download TelAnalysis Software
## 📌 What is TelAnalysis TelAnalysis helps you study your Telegram chat history. You export your data from the Telegram desktop application and load it into this dashboard. The software creates charts and reports based on your conversations. You see trends, word clouds, and sentiment results without sharing your data with external servers. ## ⚙️ System Requirements - Operating System: Windows 10 or Windows 11 - Processor: Intel Core i3 or better - Memory: 4 GB RAM minimum - Storage: 200 MB free space - Internet Connection: Required for initial setup and downloading components ## 📥 How to install Follow these steps to set up the software on your computer. 1. Visit the project website to download the latest setup file. You can find it here: Download TelAnalysis Link. 2. Locate the file in your downloads folder. 3. Double-click the file to start the installer. 4. Follow the instructions on the screen. 5. Launch the TelAnalysis dashboard from your desktop icon. ## 📤 Preparing your data The software needs your Telegram chat files to work. 1. Open Telegram Desktop. 2. Open the chat you want to analyze. 3. Click the three dots in the top right corner. 4. Select Export chat history. 5. Choose the JSON format for the output. 6. Save the folder to a known location on your PC. ## 📊 Using the dashboard Once you launch the app, follow these steps: 1. Click the "Load Data" button on the main screen. 2. Select the folder where you saved your Telegram export. 3. Wait for the software to process your text. 4. Use the navigation bar to browse different sections like sentiment, word clouds, and community maps. ## 🛡️ Privacy and Safety TelAnalysis runs on your computer. It does not send your chat content to the internet. All analysis happens within your own machine. Delete your local folders to remove your data from the software at any time. ## 🛠️ Features - View daily message volume for any chat participant. - Identify common words through visual word clouds. - Track sentiment changes over time. - Map out social networks within group chats. - Detect communities based on interaction patterns. - Export your final insights as report images. ## ❓ Frequently Asked Questions **Does this software store my login credentials?** No. TelAnalysis only reads the files you export manually. It never asks for your Telegram password or login token. **Can I analyze large chat histories?** Yes. The software handles large exports, though processing time increases with the number of messages. **Is it free?** Yes. The tool is open source and free for all users. **What formats does it support?** The software specifically expects the JSON structure generated by the Telegram Desktop export tool. ## 📝 Troubleshooting If the software fails to load your data, verify that you chose the JSON format during your Telegram export. Plain text files do not work. Ensure the directory path does not contain special characters that might cause read errors. Restart the software if the interface stops responding during a large analysis task.
